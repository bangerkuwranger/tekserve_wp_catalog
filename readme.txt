Tekserve Wordpress Catalog
====================

Integrates turn.js into a module that makes a virtual catalog element consisting of dynamically loading pages `(post_type=tekserve_catalog_page)` that can contain links to dynamically loaded products `(post_type=tekserve_catalog_product)`

Wordpress plugin to create a custom post type for Testimonials. Includes custom fields for name and organization of the quote source.

== Description ==

It's easy to create simple custom post types, and create functional dependencies in your theme while doing so. Skip it! Use this plugin to create a theme-independent custom post type for your site to maintain a collection of case studies, complete with downloadable pdfs. Easy.

== Installation ==

1. Upload the `tekserve_wp_catalog` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start creating your catalog by creating entries in the 'Catalog Pages' and 'Catalog Products' items in WordPress
4. Integrate into a page by using the 'Tekserve Catalog ' page layout template.
